08 — Conclusion

Key Findings

Insights derived from the complete ML pipeline applied to customer support ticket data.

01

Key Findings

  • Ticket Priority and Type are most influential
  • Response Duration impacts satisfaction
  • Ratings evenly spread across 1–5
  • Weak predictive signal from metadata alone
02

Best Model

Random Forest achieved the most stable CV score of 20.77%. NLP on ticket descriptions would significantly improve results.

03

Future Work

  • Apply NLP on Ticket Description
  • Hyperparameter tuning (GridSearchCV)
  • XGBoost and LightGBM experiments
  • Full Flask web app deployment
