08 — Conclusion

Key Findings

Insights derived from the complete machine learning pipeline applied to customer support ticket data.

01

Key Findings

  • Ticket Priority and Type are the most influential predictors
  • Response Duration significantly impacts satisfaction
  • Satisfaction ratings are evenly spread across 1–5
  • Dataset shows weak predictive signal from metadata alone
02

Best Model

Random Forest achieved the most stable cross-validation score of 20.77%. All models show similar accuracy suggesting NLP on ticket descriptions would significantly improve results.

03

Future Improvements

  • Apply NLP on Ticket Description text
  • Hyperparameter tuning with GridSearchCV
  • Experiment with XGBoost and LightGBM
  • Deploy as web app using Flask or FastAPI
